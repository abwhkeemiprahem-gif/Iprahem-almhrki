## ✨ الميزات التقنية للموقع | Technical Highlights

1. **🖥️ محاكي سطر الأوامر التفاعلي (CLI Terminal Simulator):** واجهة مخصصة للمطورين تتيح تنفيذ الأوامر مثل (`help`, `status`, `whoami`, `skills`) للوصول السريع إلى بيانات كفاءة النظم بطريقة برمجية مبتكرة.
2. **📊 لوحة القياسات الفورية (Telemetry Dashboard):** مخططات ورسوم بيانية تفاعلية (باستخدام Recharts) توضح معدلات التحسن الملحوظة لزمن الاستجابة ومساحة الذاكرة والإنتاجية بعد تطبيق تحسينات المهندس إبراهيم.
3. **👁️ نمط التركيز (Focus Mode):** خيار تصميمي ممتاز يتيح إخفاء المكونات العائمة والتركيز الكامل على محتوى الخدمات والتقارير لمنع التشتت البصري.
4. **📅 تقويم الاستشارات الذكي (Consultation Scheduler):** واجهة تقويم تفاعلية وسلسة تمكن العملاء من اختيار اليوم والوقت المناسب لحجز الاستشارات بشكل مباشر ومبسط.
5. **🎛️ لوحة التحكم الإدارية الاحترافية (Admin System Dashboard):** مجهزة بالكامل لتعديل وتحديث البيانات وإدارة المشروعات والخدمات مع دعم زر **تصدير كود البيانات المركزي (EXPORT MATRIX)** لتحديث مصفوفة البيانات البرمجية فوراً في ملف الإعدادات الرئيسي.

## 🛠️ التقنيات المستخدمة في البناء | Built With

- **Framework:** React 18+ with Vite (TypeScript strict type-safety).
- **Styling & Typography:** Tailwind CSS, utilizing high-end typography pairings (Inter, Space Grotesk, JetBrains Mono).
- **Animations:** `motion/react` for micro-interactions and smooth tactical card hover effects (3D tilts and glowing cursors).
- **Data Visualization:** `Recharts` for fluid, highly responsive visual metrics.
- **State Management:** Zustand-based shared store (`appStore.ts`) coordinating global events.

## 🚀 تشغيل النظام محلياً | Local Setup

تأكد من تثبيت منصة **Node.js** على جهازك قبل البدء:

### 1. تحميل المشروع (Clone)
```bash
git clone <repository-url>
cd <repository-folder>
```

### 2. تثبيت الحزم (Install)
```bash
npm install
```

### 3. إعداد المتغيرات البيئية (Environment Variables)
قم بإنشاء ملف `.env` محلي اعتماداً على `.env.example`:
```bash
cp .env.example .env
```

### 4. تشغيل خادم التطوير (Run Dev)
```bash
npm run dev
```
افتح الرابط [http://localhost:3000](http://localhost:3000) لمشاهدة واجهة المنصة بالكامل.

### 5. البناء للنشر الفعلي (Build for Production)
```bash
npm run build
```
